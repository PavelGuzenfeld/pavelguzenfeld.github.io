## Who I Am

I'm a drone avionics software engineer building safety-critical systems for autonomous flight. I design software for UAV avionics, navigation, and computer vision pipelines — with a focus on code quality, zero-overhead abstractions, and deterministic behavior on edge computing.

## What I Work With

- **Languages:** C++23, Python, CMake
- **Domains:** Drone avionics, navigation, computer vision, real-time systems
- **Frameworks:** ROS 2, PX4, GStreamer, Gazebo
- **Infrastructure:** Docker, GitHub Actions, Linux

## Open Source

I actively contribute to projects I depend on. I've submitted patches to Eigen, XGBoost, OpenCV, Fast-DDS, and dora-rs — and maintain a collection of [open-source libraries](/projects/) focused on type safety, zero-overhead abstractions, and real-time systems.
